
# Como publicar (grátis)

## Netlify
1. Acede a https://app.netlify.com e cria conta (ou login).
2. Clica **Add new site → Deploy manually**.
3. Arrasta a pasta `site_dark_portfolio` (ou ZIP) para a área de upload.
4. O link fica pronto em segundos (ex.: https://teu-site.netlify.app).

## GitHub Pages
1. Cria um repositório no GitHub chamado `portfolio-luis`.
2. Faz upload do conteúdo da pasta `site_dark_portfolio` (inclui `index.html`).
3. Vai a **Settings → Pages → Build and deployment → Deploy from a branch**.
4. Branch `main`, pasta `/ (root)`. Salva. O link sai como `https://teu-usuario.github.io/portfolio-luis`.

## Vercel
1. Acede a https://vercel.com → New Project.
2. Importa de um repositório GitHub com o conteúdo da pasta.
3. Deploy automático com link `https://...vercel.app`.
